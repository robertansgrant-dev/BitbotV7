# Claude Project Context

## Project Overview
This is a Python project template designed for collaborative AI/Developer workflows.

## Key Principles
- Clear separation of concerns
- Modular, testable code
- Comprehensive documentation
- AI-assisted development

## Development Workflow
1. **Planning**: Define requirements in requirements.txt and README.md
2. **Development**: Implement in src/ following coding standards
3. **Testing**: Add tests to tests/ directory
4. **Review**: Check against .roo/rules/
5. **Iteration**: Refactor and improve

## Quick Start
\\\ash
# Install dependencies
pip install -r requirements.txt

## File Organization
- **src/**: Source code
- **tests/**: Test files
- **docs/**: Documentation
- **.roo/**: AI workflow guidelines and prompts
- **requirements.txt**: Production dependencies
- **dev-requirements.txt**: Development dependencies

## Communication
When working with AI assistants:
- Be specific about requirements
- Provide context and examples
- Review suggested changes
- Keep feedback loop active
